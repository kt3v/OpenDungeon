# Core Access Teleporter Chamber

Core Access is the central ship node and the location of Velocity's AI chamber. The room is dominated by Velocity's crystalline core suspended in the center.

Elite Vyr and subordinate Sylph patrol this section. It is one of the most dangerous locations on Sprint and the natural climax point of the mission.

Only here can the team activate the emergency teleporter for final escape, and doing so requires all three key items.
